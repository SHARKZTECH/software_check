# software_check
 script to check for the installation of each software in the list.
 
 For example:
# The following software is installed on your computer:
- Microsoft Office
- Google Chrome
- Mozilla Firefox
- R
- WinRAR
# The following software is not installed on your computer:
- Adobe Reader
- Bitdefender
- Packet Tracer
- PuTTY
- Python
- SPSS
- RStudio
- SharpDevelop
- Code::Blocks
- Notepad++
- Dev-C++
- MySQL
- Rapid Typing
